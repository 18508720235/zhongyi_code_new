# 运控主板外设管理系统

## 功能特性

### 1. 外设查询
- **USB设备扫描**：检测所有USB设备
- **串口设备扫描**：检测所有串口设备
- **网络接口扫描**：检测以太网和WiFi接口
- **蓝牙设备扫描**：检测蓝牙适配器
- **显示设备扫描**：检测连接的显示器
- **音频设备扫描**：检测音频输入输出设备

### 2. 外设管理
- **启用/禁用功能**：控制网络接口和蓝牙设备的启用状态
- **设备详情查看**：显示设备的详细参数信息
- **协议类型筛选**：按通信协议类型筛选设备

### 3. 日志功能
- **连接日志**：记录所有设备的连接状态
- **启用日志**：记录设备的启用/禁用操作
- **日志查询**：支持按设备ID和时间范围查询

### 4. 数据导出
- **设备列表导出**：导出JSON格式的设备清单
- **统计信息**：显示各类设备的数量统计

## 安装方法

```bash
# 1. 给安装脚本执行权限
chmod +x install_peripheral.sh

# 2. 运行安装脚本
./install_peripheral.sh

# 3. 手动安装依赖（如果需要）
sudo apt install usbutils bluez net-tools
sudo pip3 install psutil pyserial tabulate
