# Ubuntu 系统管理工具

## 功能特性

### 1. 进程管理
- **查看进程列表**：实时显示系统进程信息
- **调整进程优先级**：修改进程nice值（-20到19）
- **搜索进程**：按名称搜索特定进程
- **进程日志**：记录所有进程操作历史

### 2. 存储管理
- **查看存储空间**：显示所有分区使用情况
- **分析目录大小**：统计指定目录文件数量和大小
- **清理缓存**：安全清理系统缓存文件
- **压缩备份**：创建目录的压缩备份

### 3. 日志管理
- **进程管理日志**：查看进程优先级调整记录
- **存储清理日志**：查看存储操作历史

## 安装方法

```bash
# 1. 克隆或下载项目文件
# 2. 给安装脚本执行权限
chmod +x install.sh

# 3. 运行安装脚本
./install.sh

# 或者手动安装依赖
# 使用sudo为系统级Python安装
sudo pip3 install tabulate psutil

# 或者使用apt安装（Ubuntu推荐方式）
sudo apt update
sudo apt install python3-tabulate python3-psutil
